1) To register your face to the model (face will be recorded via webcam)
```
python Add_faces.py
```

2) To record the attendance of a recognized face, press 'O'; a CSV file with the current date will be created, and the timestamp will be recorded.
```
python test.py
```

3) Attendance will be displayed here and can be downloaded in csv format
```
streamlit run app.py
```
